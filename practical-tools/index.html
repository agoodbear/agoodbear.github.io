實用工具

整合站內互動工具與原本放在 AI 人生頁面的常用工具；可直接開啟，支援嵌入的項目也能在彈出視窗中操作。

ECG 隨機考題

從自有個案與 ECG 教學部落格（Dr. Smith、LITFL、Ken Grauer）隨機抽題：題目、ECG 圖、解答、出處一次到位。支援選答、即時對答案、看詳解、一鍵下一題，並附原文出處連結。教學練習用途，ECG 圖片版權屬原作者。

立即使用

CPR 按壓速率教練

用手機鏡頭即時偵測 CPR 胸外按壓速率，判斷是否落在 100–120／分 目標區，並用紅黃綠燈、即時波形、中文語音與節拍器輔助。Pose 框上半身、框內密集光流＋自相關估主頻。純前端、影像不上傳、全程裝置端運算。教學練習輔助，非醫療器材。（iOS App 測試中）

立即使用

急性缺血性中風 治療決策矩陣

以 2026 AHA/ASA 指引為本的「時間窗 × 病人型」決策矩陣，用顏色直接代表處置（溶栓／取栓／雙抗／支持），一眼掃完整片狀況；含影像篩選標準、Penumbra/Core Mismatch 定義、TNK/tPA 劑量與到院轉院路徑，並附「互動式決策流程」（點選病人情境逐步推導建議）。內容經 OpenEvidence 對照原始試驗校訂。

立即使用

GIF PACS 雙視窗檢視器

支援左右 GIF 對照、同步翻幀、箭頭與文字標註、亮度對比調整，以及單檔 HTML 匯出保存。

立即使用

ECG 影像轉數位訊號工具

上傳 ECG 圖片後，一鍵前處理與波形擷取，可做兩點校正並匯出 CSV / JSON / PNG。

立即使用

ECG 影像標註工具

可上傳 ECG 圖檔或 PDF，支援直線、卡尺、圓形、矩形、箭頭等標記，並可匯出 JSON / PNG。

GitHub Repository

3D 心臟互動視覺化

可旋轉、縮放並選取心臟解剖結構，進行互動式 3D 解剖學習與教學展示。

立即使用

3D 心臟模型互動工具（Matt）

本地自託管 3D 心臟模型，可旋轉/縮放/平移，並以按鈕展開或閉合心臟結構。

立即使用
